# Binwalk GUI 使用说明

## 简介

Binwalk GUI 是一个为 Windows 系统设计的图形用户界面工具，用于简化 Binwalk 命令行工具的使用。本工具特别优化了在 Windows 中文环境下的显示和操作体验。

## 系统要求

- Windows 7 及以上版本
- 已安装 Python 3.6 或更高版本（如果直接运行 .py 文件）
- 或使用编译后的可执行文件（无需 Python 环境）

## 文件位置

本 GUI 工具需要与 Binwalk 可执行文件放置在同一目录下。默认位置为：
- `target\x86_64-pc-windows-gnu\debug\binwalk_gui.py` - Python 源码文件
- `target\x86_64-pc-windows-gnu\debug\binwalk.exe` - Binwalk 可执行文件

## 使用方法

### 方法一：直接运行 Python 脚本

1. 确保您的系统已安装 Python 3.6 或更高版本
2. 双击 `binwalk_gui.py` 文件
3. 或在命令行中执行：`python binwalk_gui.py`

### 方法二：使用编译后的可执行文件

1. 双击 `binwalk_gui.exe` 文件
2. 无需安装 Python 环境

## 界面功能说明

### 文件选择
- **选择文件**：点击按钮选择要分析的二进制文件
- **最近文件**：显示最近打开的文件列表

### 基本选项
- **列出扫描结果**：显示文件中的嵌入文件列表
- **提取文件**：从目标文件中提取嵌入的文件
- **详细模式**：显示更详细的扫描信息
- **递归扫描**：递归扫描提取出的文件

### 高级选项
- **指定输出目录**：设置提取文件的保存位置
- **签名文件**：使用自定义签名文件
- **搜索深度**：设置扫描的最大深度
- **启发式分析**：启用启发式文件类型识别

### 结果显示
- **日志区域**：显示命令执行的输出结果
- **复制结果**：将日志内容复制到剪贴板
- **保存日志**：将日志保存到文件

## 编译指南

我们提供了两种编译方式，推荐使用Python版本的构建脚本，它提供了更友好的界面和更详细的错误信息。

### 使用Python构建脚本（推荐）

1. 确保您的系统已安装 Python 3.6 或更高版本
2. 在 `target\x86_64-pc-windows-gnu\debug` 目录下运行：
   ```
   python build_gui.py
   ```
3. 脚本会自动检查环境、安装依赖、编译并复制可执行文件
4. 编译成功后，可执行文件将自动复制到当前目录（与binwalk.exe同目录）

### 使用批处理脚本（备用）

1. 确保您的系统已安装 Python 和 pip
2. 运行 `compile_gui.bat` 批处理脚本
3. 编译成功后，可执行文件将位于 `dist\` 目录下

## 故障排除

### 常见问题

1. **无法找到 binwalk.exe**
   - 确保 `binwalk_gui.py` 或 `binwalk_gui.exe` 与 `binwalk.exe` 在同一目录下

2. **中文显示乱码**
   - Python 版本运行时会自动设置中文字体
   - 如仍有问题，请检查系统区域设置是否为中文

3. **命令执行失败**
   - 检查目标文件是否存在且可读
   - 查看日志区域的错误信息以获取详细原因

## 注意事项

- 本工具是为了简化 Binwalk 的使用而开发的辅助工具
- 所有功能依赖于底层的 Binwalk 命令行工具
- 建议在使用前确保 Binwalk 命令行工具工作正常
